
# 🛒 Superstore Sales EDA

## Project Overview

This project presents an Exploratory Data Analysis of the Sample Superstore dataset.

The analysis focuses on sales performance, profit trends, customer segments, regional performance, and the effect of discounts on profit.

The project follows a complete data analysis workflow. It starts with data cleaning and ends with business insights supported by visualizations.

---

## Project Objectives

* Understand sales performance across different categories.
* Study profit trends across regions.
* Identify loss making products and categories.
* Analyze the impact of discounts on profit.
* Understand customer purchasing patterns.
* Build a clean dataset for analysis.
* Generate business insights from the data.

---

## Project Workflow

### 1. Data Understanding

* Examined dataset structure.
* Identified numerical columns.
* Identified categorical columns.
* Identified date columns.

### 2. Data Cleaning

* Corrected data types.
* Checked missing values.
* Removed duplicate records.
* Removed unnecessary columns.
* Standardized categorical values.

### 3. Feature Engineering

Created new features including:

* Order Month
* Order Year
* Order Day
* Delivery Days
* Profit Margin
* Discount Category
* Sales Category

### 4. Exploratory Data Analysis

Performed analysis on:

* Sales Distribution
* Profit Distribution
* Quantity Distribution
* Category Performance
* Regional Performance
* Customer Segments
* Monthly Sales Trend
* Discount vs Profit
* Correlation Heatmap

## How to Run

